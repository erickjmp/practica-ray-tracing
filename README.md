## Contido do repositorio

Neste repositorio atópase o código para a resolución dos exercicios do 1 ao 3 da parte de ray tracing. Na carpeta outputs poden atoparse imaxes correspondentes á saída obtida en cada unha das partes entregadas. 

## Versións

Cada un dos exercicios atópase marcado cun tag en cada commit definitivo:
- [Parte 1](https://github.com/erickjmp/practica-ray-tracing/releases/tag/parte-1)
- [Parte 2](https://github.com/erickjmp/practica-ray-tracing/releases/tag/parte-2)
- [Parte 3](https://github.com/erickjmp/practica-ray-tracing/releases/tag/parte-3)

## Execución

Para executar o código haberá que executar con python2 o arquivo raytracing.py:

```console
python2 raytracing.py
```
